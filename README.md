
  # Salesforce → Metronome data mappings visualization

  This is a code bundle for Visualize Salesforce Metronome Mappings. The original project is available at https://www.figma.com/design/GZxXtUdrahG5FsJyXxZe1L/Visualize-Salesforce-Metronome-Mappings.

  ## Running locally

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploy to Vercel

  Vercel auto-detects this as a Vite project.

  - Build command: `npm run build`
  - Output directory: `dist`

  ## Data

  This public repo ships with a sample CSV at `src/imports/mappings.sample.csv`. Replace it with your own mappings CSV as needed, or use the in-app CSV upload button to load a local CSV in the deployed app.
